new product ingestion pipeline automation
There's a google spreadsheet that gets updated manually every time NCE buys or adds a new product that is not on the current product catalogue. A person adds each of these fields manually.
This sheet has the following columns: product description e.g. Stainless Steel Double Basin Cocktail Sink (NCE6184), SKU (generated manually e.g. NCE6184 note, this also gest added to the product description/title), cost price, selling price,  Does product has VAT at 20% e.g. yes/no, model description, RRP, sometimes product age (this column is kinda free for all needs re-thinking), then next 3 columns are measurements w, h, d., next colum "supplier" and supplier address (again should split this).
Then a person opens the sheet and enters this information, one by one in Shopify, add if its new/used, if parcel or pallet according to the size, all of this goes as draft, because theres a period of time for the product to arrive, be cleaned, tested, photographed and upload pictures which would then make it active.
we manually say if it's new or used (candidate to come from the ingestion process) condition, e.g. used-good (again should come from ingestion) then shipping, 0 means parcel, 1 means pallet, 2 double pallet, type e.g. microwaves, vendor = Brand, collections e.g. Microwaves > Cooking Equiment > Used, tags e.g. Daewoo, Online Store template and SEO are automatically filled by shopify.
Then this info is synced every hour with QBO (exisiting integration already in place), then we go to QBO manually and complement the information.
When QBO syncs with Shopify, the only fields copied by the sync, are: Name e.g. A Board 3158 (3158 is SKU in shopify), Description e.g. New Large Oak Stain A-Board, unit price e.g. £59.50, SKU.
Then I manually have to tick "Inclusive of VAT" in QBO, update the VAT field if it's margin scheme (for used equipment) or 20% for new equipment, update the cost field, tick the "Inclusive of purchase tax" field, update "Purchase tax" field as no VAT for used equipment or 20% for new equipment, if its a recurrent supplier I find the supplier save and close, but if its new, open "prefered supplier field" choose, add new and complete all new supplier information, save and close (this comes from the last cell in spreadsheet which is currently bundled together name and address)
